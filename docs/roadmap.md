# Portal Roadmap

This roadmap outlines upcoming features, redundancy protocols, and functional expansions for the Balbodhini Pathshala Inquiry Portal.

## Phase 1: Interactive Media Logging
* **Audio & Video Vault**: Support for hosting redacted audio recordings and video testimony clips.
* **Synchronized Subtitles**: Render interactive text transcripts alongside media playbacks.

## Phase 2: Transaction Auditing
* **Interactive Bank Ledger**: An interactive ledger UI designed to display money trails, transaction IDs, bank statement logs, and proxy accounts.
* **Audit Trail Exporters**: Allow visitors to download certified CSV logs of audited fund flows.

## Phase 3: High-Availability Redundancy
* **Multi-Cloud Backups**: Auto-sync the assets vault across Amazon S3, Backblaze B2, and decentralized IPFS storage.
* **IPFS Frontend Gateway**: A decentralized mirror site pointing directly to the IPFS hashes of the evidence files.
