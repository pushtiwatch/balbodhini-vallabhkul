# Balbodhini Pathshala Inquiry Portal

A transparent, public interest disclosure vault for legal evidence.

## Quick Start
1. Fork this repo.
2. Place redacted PDFs in /assets/evidence/.
3. Update 	imeline_data.json to reflect your case.
4. Deploy via Docker.
